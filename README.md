## 📂 Repository Structure

Each problem is organized into its own self-contained folder:

To test a specific solution locally:

1. Clone this repository to your machine.
2. Navigate to the problem folder of interest.
3. Import the `Solution` class into your test script or run Python interactively.

```python
from solution import Solution

sol = Solution()
print(sol.twoSum([2, 7, 11, 15], 9)) # Output: [0, 1]
```
